Scenario: A multinational corporation operates across AWS, Azure, and GCP with inconsistent security configurations. The security team discovers cross-cloud privilege escalation opportunities through federated identity misconfigurations. The engagement requires comprehensive multi-cloud security assessment and remediation.

Question: Which THREE multi-cloud security assessment techniques provide the most critical findings for cross-cloud privilege escalation? (Select ALL that apply)
A) Federated identity trust relationship analysis across cloud providers
B) Cross-cloud network connectivity and VPC peering security assessment
C) Cloud provider security operations center (SOC) integration analysis
D) Multi-cloud compliance and regulatory requirement mapping
E) Cross-cloud logging and monitoring integration gaps identification
F) Cloud-native security tool effectiveness comparison and optimization

Answer: A, B, E

Rationale:
- A) Correct because federated identity misconfigurations often provide cross-cloud access paths that bypass traditional security boundaries
- B) Correct because cross-cloud network connectivity frequently creates unintended trust relationships and attack surfaces
- E) Correct because logging gaps across cloud providers create blind spots that attackers exploit for lateral movement
- C) Incorrect SOC integration is important for operations but doesn't typically provide direct privilege escalation opportunities
- D) Incorrect while compliance is critical, regulatory gaps don't usually create technical compromise paths
- F) Incorrect tool optimization is important for efficiency but doesn't identify security vulnerabilities

- B) Incorrect while runtime monitoring is important, it detects attacks rather than preventing supply chain compromises
- C) Incorrect RBAC and network policies are important but represent runtime controls rather than supply chain issues
- F) Incorrect platform hardening is foundational but doesn't address specific supply chain vulnerabilities

- E) Incorrect custom hunting queries are valuable but represent targeted analysis rather than comprehensive detection coverage
