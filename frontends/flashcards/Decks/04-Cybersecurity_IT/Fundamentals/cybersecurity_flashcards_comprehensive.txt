What is the primary difference between a vulnerability assessment and a penetration test?
A vulnerability assessment identifies and quantifies security vulnerabilities, while a penetration test actively exploits vulnerabilities to determine the extent of potential damage.

In Kerberos authentication, what is the Golden Ticket attack and why is it critical?
A Golden Ticket attack involves forging a Kerberos Ticket Granting Ticket (TGT) using the krbtgt account hash, allowing unrestricted access to any service in the domain. It's critical because it provides persistent domain admin access even after password changes.

When testing for SQL injection, which parameterized query approach provides the strongest protection against time-based blind injection attacks?
Using prepared statements with strongly typed parameters and explicit length validation, as this prevents both direct injection and data type manipulation attacks that can bypass simple parameterization.

What is the most effective method to prevent XXE (XML External Entity) attacks in Java applications?
Disable DTD processing entirely by setting the "javax.xml.XMLConstants.FEATURE_SECURE_PROCESSING" feature to true and using XMLInputFactory with IS_SUPPORTING_EXTERNAL_ENTITIES set to false.

In a Windows Active Directory environment, which technique provides the most stealthy lateral movement method and why?
Using DCOM/WinRM for lateral movement is most stealthy because it leverages legitimate administrative protocols that generate minimal logs compared to SMB or WMI exploitation.

What is the critical security difference between AES-256-CBC and AES-256-GCM for encrypting sensitive data at rest?
AES-256-GCM provides authenticated encryption with associated data (AEAD), offering both confidentiality and integrity protection, while CBC only provides confidentiality and requires separate MAC implementation.

When implementing secure session management, what is the most critical security consideration for session token generation?
Using cryptographically secure random number generators with sufficient entropy (minimum 128 bits) and implementing proper session invalidation on privilege changes.

What is the primary attack vector in deserialization vulnerabilities and how is it best mitigated?
The primary attack vector is malicious object injection during deserialization. Best mitigation is implementing input validation, using safe deserialization libraries, and applying digital signatures to serialized objects.

In a cloud environment, which misconfiguration poses the greatest security risk and why?
Overly permissive IAM roles with service principal credentials pose the greatest risk because they provide persistent, high-privilege access that's difficult to detect and revoke.

What is the most effective countermeasure against cross-site scripting (XSS) in modern web applications?
Implementing Content Security Policy (CSP) with strict whitelisting combined with proper output encoding and input validation provides the most comprehensive XSS protection.

When conducting a security code review, which vulnerability category is most commonly missed in custom authentication implementations?
Improper session fixation protection and inadequate logout functionality are most commonly missed, as developers focus on password validation but neglect session lifecycle management.

What is the critical security consideration when implementing OAuth 2.0 in a mobile application?

In container security, what is the most effective method to prevent container escape attacks?
Running containers as non-root users with minimal capabilities, combined with seccomp profiles and AppArmor/SELinux confinement, provides the strongest protection against escape.
