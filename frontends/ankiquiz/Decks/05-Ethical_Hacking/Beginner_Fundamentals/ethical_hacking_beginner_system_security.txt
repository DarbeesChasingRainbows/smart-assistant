What is the primary difference between Windows and Linux privilege escalation methodologies?
Windows focuses on services, permissions, and Active Directory, while Linux focuses on SUID files, kernel exploits, and configuration misconfigurations.

What is the basic command to check current user privileges in Linux?
"id" which displays the current user ID, group memberships, and privilege levels including SUID and GUID bits.

What is the difference between standard user and root privileges in Linux?
Standard users have limited system access, while root has complete control over all system resources and can modify any file.

What is the primary purpose of the "sudo -l" command in Linux privilege escalation?
To check what commands the current user can run with sudo privileges, potentially revealing privilege escalation paths.

What is the basic command to find SUID files in Linux?
"find / -perm -4000 -type f 2>/dev/null" which locates all files with SUID bit set that might be exploitable for privilege escalation.

What is the difference between SUID and SGID permissions in Linux?
SUID allows execution with file owner's privileges, while SGID allows execution with file group's privileges.

What is the primary purpose of the "linpeas" script in Linux privilege escalation?
To perform comprehensive system enumeration identifying misconfigurations, vulnerable software, and potential privilege escalation vectors.

What is the basic command to run LinPEAS in Linux?
"./linpeas.sh" which executes the comprehensive privilege escalation enumeration script to identify security weaknesses.

What is the difference between automated enumeration tools and manual privilege escalation techniques?
Automated tools quickly identify common vulnerabilities, while manual techniques allow deeper analysis and discovery of unique misconfigurations.

What is the primary purpose of kernel version enumeration in Linux privilege escalation?
To identify known kernel vulnerabilities that might be exploitable for local privilege escalation.

What is the basic command to check kernel version in Linux?
"uname -a" which displays detailed kernel information including version, architecture, and compilation details.

What is the difference between kernel exploits and userland exploits?
Kernel exploits target operating system vulnerabilities for system-level access, while userland exploits target application vulnerabilities.

What is the primary purpose of the "linux-exploit-suggester" tool?
To automatically suggest kernel exploits based on the system's kernel version and configuration.

What is the basic command to run Linux Exploit Suggester?
"./linux-exploit-suggester.sh" which analyzes the system and suggests potential kernel exploits for privilege escalation.

What is the difference between compiled kernel exploits and proof-of-concept exploits?
Compiled exploits are ready to execute, while proof-of-concept exploits demonstrate vulnerability concepts and may require modification.

What is the primary purpose of service enumeration in Linux privilege escalation?
To identify misconfigured services running with elevated privileges that might be exploitable.

What is the basic command to enumerate running services in Linux?
"ps aux" which displays all running processes with user information, potentially revealing services running as root.

What is the difference between systemd services and traditional init services?
Systemd services use modern service management with dependency handling, while init services use traditional startup scripts.

What is the primary purpose of the "systemctl" command in service enumeration?
To list, analyze, and manage systemd services, potentially identifying misconfigurations or exploitable services.

What is the basic command to list all systemd services?
"systemctl list-units --type=service --all" which displays all services including inactive ones for analysis.

What is the difference between enabled and disabled services in Linux?
Enabled services start automatically at boot, while disabled services must be started manually, affecting attack surface analysis.

What is the primary purpose of cron job enumeration in Linux privilege escalation?
To discover scheduled tasks that might run with elevated privileges or contain weak configurations.

What is the basic command to enumerate cron jobs in Linux?
"crontab -l" which displays the current user's cron jobs, potentially revealing scheduled tasks for exploitation.

What is the difference between user cron jobs and system cron jobs?
User cron jobs run with user privileges, while system cron jobs in /etc/cron* can run with elevated privileges.

What is the primary purpose of the "pspy" tool in Linux process monitoring?
To monitor running processes without root privileges, identifying cron jobs and background processes for exploitation.

What is the basic command to run pspy for process monitoring?
"./pspy64" which monitors and displays running processes and scheduled tasks in real-time.

What is the difference between process monitoring and service enumeration?
Process monitoring observes runtime behavior, while service enumeration examines static service configurations.

What is the primary purpose of file permission analysis in Linux privilege escalation?
To identify world-writable files or directories that might allow privilege escalation through file manipulation.

What is the basic command to find world-writable files in Linux?
"find / -writable -type f 2>/dev/null" which locates files that any user can modify, potentially useful for privilege escalation.

What is the difference between file permissions and directory permissions in Linux?
File permissions control access to individual files, while directory permissions control access to directory contents and traversal.

What is the primary purpose of the "find" command in Linux security enumeration?
To search for files with specific permissions, ownership, or content that might indicate security weaknesses.

What is the basic command to find files owned by specific users?
"find / -user [username] -type f 2>/dev/null" which locates all files owned by the specified user for analysis.

What is the difference between permission-based and ownership-based privilege escalation?
Permission-based exploits abuse file access rights, while ownership-based exploits target files owned by privileged users.

What is the primary purpose of password file analysis in Linux privilege escalation?
To discover weak passwords, configuration files, or credential storage that might lead to privilege escalation.

What is the difference between /etc/passwd and /etc/shadow files in Linux?
/etc/passwd contains basic user information readable by all users, while /etc/shadow contains encrypted passwords readable only by root.

What is the basic command to check if /etc/shadow is readable by non-root users?
"ls -la /etc/shadow" which displays permissions showing if regular users can read the shadow file.

What is the primary purpose of the "john" tool in Linux password cracking?
To crack password hashes using dictionary attacks, rule-based attacks, and brute force methods for privilege escalation.

What is the basic command to crack Linux password hashes with John the Ripper?
"john --wordlist=[wordlist] /etc/shadow" which attempts to crack password hashes using a dictionary attack.

What is the difference between dictionary attacks and brute force attacks in password cracking?
Dictionary attacks use predefined wordlists, while brute force attacks try all possible character combinations.

What is the primary purpose of the "hashcat" tool in GPU-accelerated password cracking?
To perform high-speed password cracking using GPU acceleration, supporting various hash types and attack methods.

What is the basic command to crack Linux hashes with Hashcat?
"hashcat -m 1800 -a 0 [hash_file] [wordlist]" which performs dictionary attacks against Linux password hashes.

What is the difference between CPU and GPU password cracking?
GPU cracking uses graphics processors for massively parallel processing, while CPU cracking uses general-purpose processors.

What is the primary purpose of network configuration enumeration in Linux privilege escalation?
To discover network interfaces, routes, and connections that might reveal additional attack surfaces or misconfigurations.

What is the basic command to enumerate network interfaces in Linux?
"ip addr show" which displays all network interfaces with IP addresses and configuration details.

What is the difference between interface enumeration and connection enumeration?
Interface enumeration examines network adapters and configuration, while connection enumeration shows active network connections.

What is the primary purpose of the "netstat" command in Linux network enumeration?
To display active network connections, listening ports, and network statistics for security analysis.

What is the basic command to show all network connections in Linux?
"netstat -tulnp" which displays TCP and UDP connections with process information for security analysis.

What is the difference between TCP and UDP connection analysis?
TCP analysis examines connection-oriented protocols, while UDP analysis examines connectionless protocols for potential vulnerabilities.

What is the primary purpose of environment variable enumeration in Linux privilege escalation?
To discover sensitive information, configuration details, or potential privilege escalation paths stored in environment variables.

What is the basic command to display environment variables in Linux?
"env" or "printenv" which displays all environment variables for the current user and process.

What is the difference between user and system environment variables?
User variables apply to specific user sessions, while system variables apply globally to all users and processes.

What is the primary purpose of the "sudoers" file analysis in Linux privilege escalation?
To identify misconfigured sudo rules that might allow command execution with elevated privileges.

What is the basic command to check sudoers file permissions?
"ls -la /etc/sudoers" which displays permissions and ownership of the sudoers file for security analysis.

What is the difference between sudo configuration analysis and direct privilege escalation?
Sudo analysis examines permission configurations, while direct escalation attempts to exploit existing misconfigurations.

What is the primary purpose of Windows user enumeration in privilege escalation?
To identify user accounts, group memberships, and privilege levels that might lead to escalation opportunities.

What is the basic command to enumerate users in Windows?
"net user" which displays all local user accounts on the Windows system for privilege escalation analysis.

What is the difference between local users and domain users in Windows?
Local users exist only on the specific machine, while domain users exist across the Active Directory domain.

What is the primary purpose of the "whoami" command in Windows privilege escalation?
To display the current user and privilege level, helping understand the starting point for privilege escalation.

What is the basic command to check current privileges in Windows?
"whoami /priv" which displays all privileges held by the current user, potentially identifying escalation opportunities.

What is the difference between user privileges and group privileges in Windows?
User privileges are assigned directly to accounts, while group privileges are inherited through group membership.

What is the primary purpose of the "net localgroup" command in Windows enumeration?
To enumerate local groups and their members, potentially identifying privilege escalation through group membership.

What is the basic command to enumerate administrators group in Windows?
"net localgroup Administrators" which displays all members of the local administrators group.

What is the difference between built-in groups and custom groups in Windows?
Built-in groups have predefined privileges, while custom groups can be configured with specific permissions and memberships.

What is the primary purpose of service enumeration in Windows privilege escalation?
To identify services running with elevated privileges that might be exploitable through configuration weaknesses.

What is the basic command to enumerate services in Windows?
"sc query" which displays information about all services running on the Windows system.

What is the difference between automatic and manual service startup types?
Automatic services start at boot, while manual services require explicit startup, affecting attack surface analysis.

What is the primary purpose of the "accesschk" tool in Windows privilege analysis?
To analyze file, registry, and service permissions to identify potential privilege escalation opportunities.

What is the basic command to check file permissions with accesschk?
"accesschk.exe -w [file_path]" which checks write permissions on the specified file or directory.

What is the difference between file permissions and registry permissions in Windows?
File permissions control access to files and folders, while registry permissions control access to registry keys and values.

What is the primary purpose of scheduled task enumeration in Windows privilege escalation?
To discover scheduled tasks that might run with elevated privileges or contain weak configurations.

What is the basic command to enumerate scheduled tasks in Windows?
"schtasks /query /fo LIST" which displays all scheduled tasks on the Windows system.

What is the difference between user and system scheduled tasks?
User tasks run with user privileges, while system tasks can run with system or elevated privileges.

What is the primary purpose of the "powershell" command in Windows privilege escalation?
To perform advanced enumeration and exploitation using PowerShell's powerful scripting capabilities.

What is the basic command to check PowerShell execution policy?
"Get-ExecutionPolicy" which displays the current PowerShell execution policy affecting script execution.

What is the difference between PowerShell and CMD in Windows security testing?
PowerShell provides advanced scripting and .NET access, while CMD offers basic command-line functionality.

To identify common Windows privilege escalation vectors including misconfigurations, vulnerable services, and weak permissions.

What is the difference between automated PowerUp enumeration and manual Windows analysis?

What is the basic command to run WinPEAS in Windows?
".\winPEAS.exe" which executes the comprehensive Windows privilege escalation enumeration script.

What is the difference between WinPEAS and manual Windows enumeration techniques?
WinPEAS provides automated comprehensive analysis, while manual techniques focus on specific vulnerability areas.

What is the primary purpose of registry analysis in Windows privilege escalation?
To discover sensitive information, weak permissions, or configuration issues in the Windows registry.

"Get-ItemProperty -Path [registry_path]" which retrieves values and permissions from specified registry keys.

What is the difference between registry keys and registry values in Windows security?

To query, modify, and analyze registry keys and values from the command line for security assessment.

What is the basic command to query registry permissions with reg?

What is the difference between registry enumeration and file system enumeration in Windows?

What is the primary purpose of process enumeration in Windows privilege escalation?
To identify processes running with elevated privileges or containing vulnerable configurations.

What is the basic command to enumerate processes in Windows?
"tasklist /v" which displays all running processes with detailed information including user context.

What is the difference between process enumeration and service enumeration in Windows?
Process enumeration examines running executables, while service enumeration examines background service configurations.

What is the primary purpose of the "handle" tool in Windows process analysis?
To enumerate file handles held by processes, potentially identifying access to sensitive files or resources.

What is the basic command to check handles with the handle tool?
"handle.exe -p [process_id]" which displays all file handles held by the specified process.

What is the difference between handle analysis and process analysis in Windows security?

What is the primary purpose of the "procmon" tool in Windows monitoring?

What is the difference between real-time monitoring and static analysis in Windows security?

What is the primary purpose of the "autoruns" tool in Windows security analysis?

What is the basic command to run autoruns from command line?

What is the difference between startup programs and services in Windows privilege escalation?

What is the primary purpose of DLL hijacking in Windows privilege escalation?
To replace legitimate DLLs with malicious ones to achieve code execution with the privileges of the loading application.

What is the basic technique to test for DLL hijacking?
Place a malicious DLL in the application's search path to see if it gets loaded instead of the legitimate DLL.

What is the difference between DLL hijacking and service exploitation in Windows?
DLL hijacking exploits library loading mechanisms, while service exploitation exploits service configuration weaknesses.

What is the primary purpose of the "accesschk64" tool in modern Windows privilege analysis?

What is the basic command to analyze directory permissions with accesschk64?

What is the difference between 32-bit and 64-bit Windows privilege escalation?

What is the primary purpose of UAC (User Account Control) bypass techniques in Windows privilege escalation?
To circumvent UAC restrictions to gain elevated privileges without proper user consent or interaction.

What is the basic technique to test UAC bypass?
Attempt to execute privileged operations through trusted binaries or configuration weaknesses to bypass UAC restrictions.

What is the difference between UAC bypass and direct privilege escalation in Windows?
UAC bypass circumvents protection mechanisms, while direct escalation exploits existing misconfigurations or vulnerabilities.
